*1 Use data.sql to create tables in database or you can create them in your own way
*2 Change the following fields in /application/config/database.php to match the settings in your environment:
	'username' => 'root',
	'password' => '',
	'database' => 'db_login&sign_up'(no need to change if you use data.sql in step 1)
*3 Change your folder name to "Login-Sign_up" (if not)
*4 Login page:
  http://localhost/Login-sign_up/index.php/Auth/login
   Sign up page:
  http://localhost/Login-Sign_up/index.php/Auth/Sign_up
